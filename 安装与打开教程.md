# 《每日专注》安装与打开教程


## 你需要先装一样东西：Node.js

这个应用要在电脑上跑起来，需要先装一个叫 **Node.js** 的小环境（装一次就行）。

### 方法一：官网下载（推荐）

1. 用浏览器打开：**https://nodejs.org**
2. 页面上找 **LTS**（长期支持版），点下载 **Windows 安装包**（`.msi`）。
3. 双击安装，一路「下一步」。
4. **重要**：安装时如果有 **“Add to PATH”** 或 **“自动添加环境变量”**，请**勾选**。
5. 装完后，**关掉所有已经打开的** 黑色命令行窗口、PowerShell、以及 **“ VS Code**，再重新打开。  
   （不重启的话，有时会提示找不到 `npm` 命令。）

### 方法二：有 Windows 应用商店时

1. 打开 **Microsoft Store**，搜索 **Node.js LTS**，安装。
2. 同样装完后**重启 窗口** 再试。

---

## 怎么确认装好了？

1. 按键盘 `Win + R`，输入 `cmd`，回车，打开**命令提示符**。
2. 输入下面两行，每行输完按回车：

```text
node -v
npm -v
```

如果各自显示出一串版本号（比如 `v18.x.x`），就说明装好了。

---

## 打开《每日专注》这个项目


### 第一步：安装依赖（第一次必做）

在终端里输入，回车：

```bat
npm install
```

---

### 第二步：启动网页

在终端里输入，回车：

```bat
npm run dev
```

终端里会出现几行字，其中有一行是本地地址，一般是：

```text
http://localhost:5173
```

**用浏览器打开这个地址**（Chrome、Edge 都可以）。

## 在软件里可以做什么（超简版）

| 底部按钮 | 作用 |
|----------|------|
| 收集箱 | 随手记事情，再认领到今天 |
| 今日 | 今天选最多 3 件事来做 |
| 专注 | 计时，帮自己专心一段 |
| 打卡 | 自己设几个「小习惯」，每天勾选，月历上会变色 |

右上角 **导出 / 导入** 可以备份数据，换电脑前建议先 **导出** 存好。
